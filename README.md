**Kanastra Backend Challenge**

## Overview
This repository contains the backend solution for the Kanastra backend challenge. It is built using Django, PostgreSQL, and Docker.

## Technologies Used
- Django
- PostgreSQL
- Docker

## Requirements
- Python: latest
- PostgreSQL: latest
